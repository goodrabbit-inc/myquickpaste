Clipee

よく使う文字列を保存。
タップですぐコピー。

スマートフォン向けの軽量ツール。カテゴリ別に定型文を登録し、 一覧からタップするだけでクリップボードへコピーできます。

Android / iOS 向けモバイルアプリ

主な機能

  • タップでコピー — 一覧の行をタップすると内容がクリップボードにコピーされます。
  • 長押しプレビュー — 行を長押しすると、タイトルと本文をその場で確認できます。
  • カテゴリ切り替え — フリーワード・住所・本人など、用途別に一覧を分けて管理できます。
  • 本人カード — 本人・勤務先の連絡先をカード形式で登録し、項目ごとにコピーできます。
  • バックアップ — JSON 形式でエクスポート / インポートが可能です。

データとプライバシー

登録した文字列、本人情報、設定はすべて端末内にのみ保存されます。 データが外部サーバーへ送信されることはありません。
