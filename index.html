MyQuickPaste

よく使う文章とコピー履歴を、瞬時に貼り付け

毎日のコピー＆ペーストを効率化する、軽量な Windows 用ツールです。 定型文の登録、クリップボード履歴、クイック変換で作業を快適にします。

Microsoft Store でダウンロード

Microsoft の審査を受けた安全なアプリです（14日間全機能試用あり）

ホーム サポート リリースノート プライバシーポリシー

主な機能

  • 定型文スロット — よく使う文章をタブごとに登録（無料版は10スロット、Proで拡張）
  • クリップボード履歴 — 最大20件を自動保存（Pro）
  • クイックアクセス — 日付・時刻・大文字小文字変換・履歴呼び出し
  • テーマ — Light / Dark / さくら色 など
  • 多言語 — 17言語以上の UI と便利リスト

こんな場面で

オフィス・メール

署名や定型文、よくある返信をワンクリックで貼り付け。

Excel

よく使う関数や固有名詞を登録して、セルへすぐ挿入。

カスタマーサポート

FAQ や挨拶文をカテゴリ別に登録し、返信を短時間で。

ゲーム・SNS

定型の挨拶や URL を素早くチャットへ。

Pro 版について

本体アプリは無料です。スロット無制限・履歴・インポート/エクスポートなどの Pro 機能は、 Microsoft Store のアドオン（買い切り）でご利用いただけます。価格は Store 画面でご確認ください。
